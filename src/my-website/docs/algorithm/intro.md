# 介紹
在電腦科學中，演算法（Algorithm）是指解決特定問題或執行任務的一套有限、明確且可執行的步驟指令。 

以下是演算法的核心特徵與應用： 

## 1. 五大基本特性 (Finiteness & Logic) 
* 輸入（Input）：有零個或多個外部輸入。
* 輸出（Output）：至少有一個輸出結果。
* 明確性（Definiteness）：每個步驟必須清晰無歧義。
* 有限性（Finiteness）：必須在執行有限步後結束，不會陷入無窮迴圈。
* 有效性（Effectiveness）：每個步驟都必須夠基礎，能用筆紙紙計算出來。 

## 2. 常見的演算法類型 
* 排序演算法（Sorting）：如快速排序（Quick Sort）、合併排序（Merge Sort），用於排列資料順序。
* 搜尋演算法（Searching）：如二分搜尋法（Binary Search），在資料中尋找特定目標。
* 動態規劃（Dynamic Programming）：將複雜問題分解為重疊的子問題以節省計算時間。
* 機器學習演算法：如線性回歸、神經網路，讓電腦從數據中學習模式。 

## 3. 衡量標準：複雜度分析 
評估一個演算法的好壞通常看： 
* 時間複雜度（Time Complexity）：執行需要多久？常用 Big O 符號（如 \(O(n)\), \(O(logn)\)）表示。
* 空間複雜度（Space Complexity）：執行需要多少記憶體？
