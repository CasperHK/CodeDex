# 作業系統
作業系統（OS）是管理電腦硬體、軟體資源的的核心程式，充當使用者與硬體之間的橋樑，負責資源分配、檔案管理、設備控制等，讓使用者能順暢執行應用程式（如Windows、macOS、Linux、Android），是電腦能正常運作的基礎。 

## 主要功能
* 資源管理：分配和管理CPU、記憶體、儲存空間、周邊設備（鍵盤、滑鼠）等。
* 使用者介面：提供圖形化界面 (GUI) 或命令行界面 (CLI)，讓使用者與電腦互動。
* 程式執行：控制和協調應用程式的執行，提供執行環境。
* 文件系統：管理檔案和資料夾的儲存、存取與組織。
* 安全性：保護系統免受惡意軟體和未經授權的存取。 

## 為什麼需要作業系統
* 簡化複雜性：讓使用者和程式設計師不必直接操作硬體，透過作業系統的API就能使用硬體功能。
* 多工處理：管理多個程式同時運行，提供公平有效的資源分配。 

## 常見的作業系統
桌面/伺服器：Windows (Windows 10, 11), macOS, Linux (Ubuntu, CentOS)。
行動裝置：Android, iOS。
嵌入式：用於智慧家電、車載系統等。
