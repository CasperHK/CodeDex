# Linux
Linux 是一個受歡迎的開源作業系統內核，也是許多不同發行版（Distro）的基礎。由於其穩定性、安全性和靈活性，它被廣泛用於伺服器、桌面和嵌入式系統。

## 常見的 Linux 發行版
- **Ubuntu**: 以易用性著稱，適合初學者和桌面使用者。
- **Debian**: 以穩定性和安全性聞名，常用於伺服器環境。
- **Fedora**: 由 Red Hat 支持，包含最新的軟體和技術。
- **CentOS**: 基於 Red Hat Enterprise Linux，適合企業級應用。
- **Arch Linux**: 提供高度自訂化的安裝選項，適合進階使用者。
- **Kali Linux**: 專門為滲透測試和安全分析設計的發行版。
- **Deepin**: 以美觀的桌面環境和易用性為特色。
- **Manjaro**: 基於 Arch Linux，但更易於安裝和使用。
- **UOS**: 由中國開發，針對國內市場進行優化。
- **Oracle Linux**: 由 Oracle 公司開發，適合企業應用。
- **Rocky Linux**: 由 Rocky Enterprise Software Foundation 開發，是 CentOS 的替代品。
- **TALOS Linux**: 一個專為 Kubernetes 從零開始構建的現代化、安全且不可變的 Linux 發行版。
- **Alpine Linux**: 一個輕量級的 Linux 發行版，適合嵌入式系統和容器環境。
