# Python 機器學習函式庫概覽

在 Python 生態系統中，有許多強大且廣泛使用的機器學習函式庫，這些函式庫涵蓋了從資料前處理、模型建立到評估與部署的各個階段。以下是一些主要的 Python 機器學習函式庫：

1. **Scikit-learn**
   - 功能：提供簡單且高效的工具，用於資料挖掘和資料分析，涵蓋分類、回歸、聚類等多種機器學習任務。
   - 特點：易於使用，適合初學者和中小型專案，與 NumPy 和 Pandas 無縫整合。
   - 官網：[https://scikit-learn.org/](https://scikit-learn.org/)

2. **TensorFlow**
   - 功能：由 Google 開發的開源深度學習框架，支援大規模機器學習和神經網路模型的建立與訓練。
   - 特點：高度靈活，適合構建複雜的深度學習模型，支援分散式運算。
   - 官網：[https://www.tensorflow.org/](https://www.tensorflow.org

3. **PyTorch**
   - 功能：由 Facebook 開發的深度學習框架，以動態計算圖聞名，適合研究和生產環境。
   - 特點：易於調試和擴展，廣泛應用於學術研究和工業界。
   - 官網：[https://pytorch.org/](https://pytorch.org/)

4. **Keras**
   - 功能：高階神經網路 API，能夠在 TensorFlow、Theano 和 CNTK 上運行，簡化深度學習模型的構建過程。
   - 特點：使用者友好，適合快速原型設計和實驗。
   - 官網：[https://keras.io/](https://keras.io/)

5. **XGBoost**
   - 功能：高效的梯度提升決策樹（GBDT）實現，廣泛用於結構化資料的分類和回歸任務。
   - 特點：速度快，性能優異，常用於競賽中。
   - 官網：[https://xgboost.readthedocs.io/](https://xgboost.readthedocs.io/)

6. **LightGBM**
   - 功能：由微軟開發的梯度提升框架，專為高效處理大規模資料而設計。
   - 特點：速度快，內存佔用低，支援類別特徵。
   - 官網：[https://lightgbm.readthedocs.io/](https://lightgbm.readthedocs.io/)

7. **CatBoost**
   - 功能：由 Yandex 開發的梯度提升決策樹框架，特別適合處理類別特徵。
   - 特點：易於使用，性能優異，減少過擬合。
   - 官網：[https://catboost.ai/](https://catboost.ai/)

8. **Hugging Face**
   - 功能：專注於自然語言處理（NLP）的深度學習函式庫，提供大量預訓練模型和工具。
   - 特點：易於使用，支援多種 NLP 任務，如文本分類、命名實體識別和文本生成。
   - 官網：[https://huggingface.co/](https://huggingface.co/)

9. **JAX**
   - 功能：由 Google 開發的高效數值計算庫，支援自動微分和 JIT 編譯。
   - 特點：易於使用，支援多種機器學習模型的快速開發與部署。
   - 官網：[https://jax.readthedocs.io/](https://jax.readthedocs.io/)

10. **Pandas**
   - 功能：強大的資料操作和分析函式庫，提供高效的資料結構如 DataFrame。
   - 特點：廣泛用於資料前處理和探索性資料分析，與其他機器學習函式庫無縫整合。
   - 官網：[https://pandas.pydata.org/](https://pandas.pydata.org/)

11. **NumPy**
   - 功能：提供支援大型多維陣列和矩陣運算的基礎函式庫。
   - 特點：是科學計算和機器學習的基石，與其他機器學習函式庫緊密結合。
   - 官網：[https://numpy.org/](https://numpy.org/)

12. **AutoGluon**
   - 功能：由亞馬遜開發的自動機器學習（AutoML）函式庫，支援圖神經網路和深度學習模型的自動化構建與訓練。
   - 特點：簡化模型選擇和超參數調整過程，適合快速開發和部署。
   - 官網：[https://autogluon.mxnet.io/](https://auto.gluon.ai/)

13. **Perpetual ML Suite**
   - 功能：提供全面的機器學習工具集，支援模型訓練、部署和監控。
   - 特點：易於使用，適合企業級應用，支援多種機器學習任務。
   - 官網：[https://www.perpetual.ml/](https://www.perpetual.ml/)

這些函式庫各有其優勢和適用場景，選擇合適的工具將有助於提升機器學習專案的效率和效果。根據專案需求，可以單獨使用某個函式庫，或結合多個函式庫來達成最佳效果。
